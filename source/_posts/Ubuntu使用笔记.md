# 前言
使用Ubuntu过程中，经常会遇到各种奇怪的问题，有些可以在网上搜索解决，有些需要不断的尝试，这些方法记录在这里。本文的系统是Ubuntu 17.10 mate。

 
# 安装python-webkit
17.10系统里没有python-webkit单独的包，可以通过安装python-gtk2解决。
```bash
sudo apt-get install pthon-gtk2
```
